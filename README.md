# Описание pet-проекта:

Отображение данных погоды по городам, поиск которых осуществляется с помощью SearchBar из выезжающего снизу ViewController.

```
Применение:
MVC
MapKit
CoreLocation
RestAPI
KeyboardNotification
UIViewPropertyAnimator
xib 
```
